SISTEMA INTELIGENTE DE MONITORAMENTO DE SAÚDE DE PACIENTES EM LEITOS HOSPITALARES

Este repositório contém o código-fonte funcional e a documentação do Trabalho de Conclusão de Curso (TCC) sobre o desenvolvimento de um sistema inteligente para monitoramento de pacientes internados em leitos hospitalares.

Projeto_TCC_Versao_Final_2025.docx: Documento final do TCC, detalhando todo o projeto, metodologia, resultados e conclusões.

Codigo_Funcional_Projeto_TCC.txt: Código final do sistema e pronto para usar.
